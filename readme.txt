このスクリプトを実行する為には、npmで「socket.io」パッケージの追加インストールが必要

・run.bat
	Node.jsサーバーを起動する

・othelloクライアント.url
	クラアントを起動する
	2画面以上の起動でゲームが開始する
	1Playモードも作成したが、現状は適切に動作せず


・server.js
	Node.jsのサーバー側のメイン・スクリプト

・othello.htm
	クライアントアクセスに表示する内容を記述したHTMLファイル

・othello.js
	クライアント側の動的動作を行う為のJavaScript

・style.css
	クライント表示内容を調整する為のStyleSheet
